# QA Checklist

QA Engineer uses this for acceptance testing.

## AC Coverage
- [ ] Every acceptance criterion tested
- [ ] PASS/FAIL recorded per AC item
- [ ] Evidence screenshot for each AC

## Edge Cases
- [ ] All story edge cases tested
- [ ] Error states produce user-friendly messages
- [ ] Boundary values handled correctly

## Regression
- [ ] Core user journeys from previous stories verified
- [ ] No regressions introduced

## Browser Evidence
- [ ] Console errors captured
- [ ] Network requests verified
- [ ] Traces saved to .stagix/qa/evidence/
